# Training Planner (План тренировок)

**Автор**: Гордеев Семён
**Вариант:** №1 Training Planner
**Дата сдачи:** 27.05.2026

## Описание

Training Planner - это GUI-приложение для планирования и отслеживания тренировок. Приложение позволяет добавлять, просматривать, фильтровать и удалять тренировки с сохранением данных в JSON формате.

## Основные функции

- Добавление тренировок с указанием даты, типа и длительности
- Просмотр списка всех тренировок в табличном виде
- Фильтрация по типу тренировки и диапазону дат
- Сохранение и загрузка данных в/из JSON файла
- Валидация вводимых данных
- Удаление отдельных записей

## Требования

- Python 3.6 или выше
- Библиотека tkinter (входит в стандартную поставку Python)
